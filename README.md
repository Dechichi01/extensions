TFGCo Go Extensions
===================

This package contains the common extensions we use in our projects.

### Extensions
* Postgres
* Statsd
* Kafka Consumer
* Kafka Producer

### Dependencias
* [librdkafka](https://github.com/edenhill/librdkafka)
